# চলো পড়তে বসি (Cholo Porte Boshi)

An open educational web app following the Bangladesh national curriculum. Provides free, login-free access to structured courses and lessons. Admin-managed via Firebase, hosted on GitHub Pages. Built for learners, by learners.
